This page is designed in the "Column Drop" responsive pattern with elements stacking one on top of the other at its most narrow view point. CSS4 media query elements are utilized to established stylistically necessary breakpoints, while the Bootstrap grid system creates the necessary columns and rows to structurally support the site.

Bootstrap

Bootstrap's gird template elements are utilized for the basic form and structure of this demo page. Additionally, the "img-responsive" element was leveraged to dynamically resize images to ensure a truly responsive, and mobile first approach to this demo site.

HTML5 & CSS4

This demo details fundamental mastery of both HTML5 and CSS4. The entire structure was built in HTML5, while the styling is in a separate CSS file. Media queries are used for each established breakpoints built around the "Mobile First" paradigm. Font styling is the "Dosis" text from Google fonts. .

Responsive Design

This page utilizes the "Column Drop" responsive design pattern. At the sights narrowest viewport each of the sites elements will stack one on top of the other. As the site widens to the first breakpoint the elements realign to be next to each other up until 950 pixels at which point the site content will stop expanding, and static margins are added. Key break points can be observed at the following widths:

992px - Each of the "Front-end Skillset" models realign to be right next to each other, the arrow image on the right side of the header disappears.

950px- Adds static margins, prevents elements from stretching any further.

387px - h3 headers disappear all images are given a 10% margin buffer.

259px- Arrow image and h2 disappears, h1 becomes center aligned.
